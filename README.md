# Spaceship Game
Spaceship that flies in space and shoot also a practice project in pygame

## Covered Topics
- Drawing surfaces
- Rectangles
- Movements and delta time
- Getting input from the user
- Working with time (Time intervals)
- Drawing other things (Rect, Cirlce and etc.)
- Masks, Transforming surfaces, Animations and Sounds
